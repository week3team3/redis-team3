# redis project

## branch 생성 및 git 연결

1. < > Code 🔻 누르기
2. clone 받을 주소 복사
3. 터미널에 입력

   $ git clone https://github.com/week3team3/redis-team3.git

4. clone받은 폴더로 이동 cd 현재폴더명
5. 브랜치 생성
   $ git checkout -b 브랜치명
6. 변경된 브랜치에서 작업

7. 작업사항 push 방법
   $ git add .
   $ git commit -m "커밋메세지 작성(작업내용)"
   $ git push origin 브랜치명
8. 깃허브 팀 레포지터리의 'Pull requests'에서 pr사항 확인 후 이상 없으면 merge
